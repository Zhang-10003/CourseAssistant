# CourseAssistant - 课堂互动教学系统

## 项目简介

CourseAssistant 是一个基于 FastAPI 和 Vue 3 开发的课堂互动教学系统，旨在提升课堂教学的互动性和效率。系统支持教师发布作业、管理课程，学生提交作业、参与互动，以及基于 AI 的智能问答和出题功能。

## 技术栈

### 后端技术

- **FastAPI**: 高性能的 Python Web 框架
- **SQLAlchemy**: ORM 数据库工具
- **MySQL**: 关系型数据库
- **JWT**: 身份认证
- **FastAPI-Mail**: 邮件服务
- **APScheduler**: 任务调度
- **OpenAI SDK**: 大模型 API 调用

### 前端技术

- **Vue 3**: 前端框架
- **Vite**: 构建工具
- **Element Plus**: UI 组件库
- **Axios**: HTTP 客户端

## 核心功能

### 教师功能

- 课程管理：创建、编辑、删除课程
- 作业管理：发布、批改、管理作业
- 智能出题：基于知识点自动生成考试题目
- 课件上传：将课件上传到知识库，用于智能问答

### 学生功能

- 课程参与：查看课程信息，参与课堂互动
- 作业提交：提交作业，查看批改结果
- 智能问答：向系统提问，获取基于课程内容的回答

### 管理员功能

- 用户管理：管理教师和学生账号
- 系统配置：配置系统参数和 API 密钥

### AI 功能

- 智能问答：基于课程内容回答学生问题
- 智能出题：根据知识点自动生成考试题目
- 文档处理：处理上传的课件，提取知识点

## 安装与运行

### 后端安装

1. 克隆项目
   ```bash
   git clone <项目地址>
   cd CourseAssistant/backend
   ```
2. 创建虚拟环境
   ```bash
   conda create -n backend python=3.10
   conda activate backend
   ```
3. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```
4. 配置环境变量
   - 在 `backend/setting/__init__.py` 中配置数据库连接和邮件服务
5. 启动服务
   ```bash
   uvicorn main:app --reload
   ```

### 前端安装

1. 进入前端目录
   ```bash
   cd CourseAssistant/frontend
   ```
2. 安装依赖
   ```bash
   npm install
   ```
3. 启动开发服务器
   ```bash
   npm run dev
   ```

## API 接口

系统提供以下主要 API 接口：

- **认证接口**: 登录、注册、刷新 token
- **作业接口**: 创建、获取、更新、删除作业
- **题目接口**: 生成题目、获取题目
- **教师接口**: 教师相关操作
- **学生接口**: 学生相关操作

## 系统特色

1. **智能互动**：基于 RAG 技术，实现智能问答和出题功能
2. **实时反馈**：作业提交后立即获得反馈
3. **易于使用**：直观的用户界面，操作简单
4. **可扩展性**：模块化设计，易于添加新功能
5. **安全可靠**：JWT 认证，数据加密
